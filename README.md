## Karunya School Website

This project is a responsive school website built using **React and Vite**.
It showcases campus facilities, sports activities, admissions information, branding sections, and testimonials.

## Technologies Used

* React
* Vite
* JavaScript
* CSS

## Installation

Clone the repository:

git clone https://github.com/jagadeesh3276/your-repository-name.git

Install dependencies:

npm install

Run the development server:

npm run dev

## Build for Production

npm run build

The optimized production files will be generated inside the **dist** folder.

## Deployment

The project is deployed using **GitHub Pages**.

npm run deploy
